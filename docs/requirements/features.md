# 機能一覧
本ドキュメントは、英単語学習Webアプリケーションにおける機能一覧を整理し、画面要件・API設計・実装作業の指針とすることを目的とする。

---

## 1. 認証機能

### 1-1. ログイン

- 機能要件
  ユーザーがメールアドレスとパスワードを用いてログインする
- 対象画面
  ログイン画面
- API
  POST /auth/login
- 備考
  利用者は1名想定

---

## 2. 学習モード管理

### 2-1. 学習モード選択

- 機能概要
  学習モード（カスタマイズ/プリセット）
- 対象画面
  モード選択画面
- API
  なし
- 備考
  選択したモードにより出題対象が切り替わる

---

## 3. 単語学習（カスタマイズモード）

### 3-1. 単語一覧表示

- 機能概要
  登録済み単語を一覧表示する
- 対象画面
  単語一覧画面
- API
  GET /words
- 備考
  タグ・暗記フラグによる絞り込みを行う

### 3-2. 単語登録

- 機能概要
  英単語を新規登録する
- 対象画面
  単語登録画面
- API
  POST /words
- 備考
  辞書APIから意味・例文を自動取得する
  重複登録は不可

### 3-3. 単語詳細表示

- 機能概要
  単語の詳細情報を表示する
- 対象画面
  単語詳細画面
- API
  GET /words/{id}
- 備考
  意味・例文・タグ・暗記フラグを表示

### 3-4. 単語編集

- 機能概要
  投稿済み単語の情報を編集する
- 対象画面
  単語詳細画面
- API
  PUT /words{id}
- 備考
  カスタマイズモードのみ可能

### 3-5. 単語削除

- 機能概要
  登録済み単語を削除する
- 対象画面
  単語詳細画面
- API
  DELETE /words{id}

---

## 4. 単語管理（プリセットモード）

### 4-1. プリセット単語一覧表示

- 機能概要
  システム提供の単語帳を表示する
- 対象画面
  単語一覧画面
- API
  GET /preset/words
- 備考
  単語内容は編集不可

---

## 5. クイズ機能

### 5-1. クイズ開始

- 機能概要
  選択した単語帳をもとにクイズを開始する
- 対象画面
  単語一覧画面
- API
  GET /quiz/start

### 5-2. 出題取得

- 機能概要
  出題条件に基づき問題を取得する
- 対象画面
  クイズ画面
- API
  GET /quiz/next
- 備考
  出題頻度・暗記フラグを考慮

### 5-3. 回答送信

- 機能概要
  回答を送信し、正誤判定を行う
- 対象画面
  クイズ画面
- API
  POST /quiz/answer

---

## 6.学習結果・履歴

### 6-1. 学習結果表示

- 機能概要
  クイズ結果を表示する
- 対象画面
  学習結果画面
- API
  GET /quiz/result

### 6-2. 学習履歴保存

- 機能概要
  単語ごとの正誤履歴を保存する
- 対象画面
  クイズ画面
- API
  POST /quiz/history
- 備考
  モードごとに履歴を分離管理する
