# 外部インターフェース一覧

## 1. 外部サービス一覧

### Jisho.org API

- 種別
  辞書API
- 用途
  英単語登録時の意味・例文取得

## 2. 利用概要

### Jisho.org API

- 利用タイミング
  単語登録時
- 利用目的
  意味・例文の自動取得
- 連携方式
  HTTP REST API
- データ保存
  取得結果を WORDS テーブルに保存
- 再取得
  原則行わない（手動編集可）

## 3. 外部IFに関する前提・制約

- Jisho.org APIが利用不可の場合
  → 手動入力による単語登録を許可
- レート制限や仕様変更については考慮しない
- 外部APIエラーはユーザーに簡易メッセージで通知する
