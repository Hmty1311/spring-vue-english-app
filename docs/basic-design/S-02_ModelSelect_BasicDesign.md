# S-02 モード選択画面 基本設計書

---

## 1. 画面概要

ログイン後に表示される機能選択画面。各機能画面から本画面へ戻ることができるハブ画面とする。

---

## 2. 利用API

| API ID | API名 | 概要 |
| -- | -- | -- |
| AUTH-02 | ログアウトAPI | セッションを破棄する |

---

## 3. 表示項目

### 3-1. 一覧表示項目一覧

| 表示名 | 項目名 | 表示 | 備考 |
| --- | -- | -- | -- |
| 単語管理ボタン | wordManageButton | ○ | 単語一覧画面へ遷移 |
| クイズモードボタン | quizModeButton | ○ | クイズ画面へ遷移 |
| ログアウトボタン | logoutButton | ○ | ログイン画面へ戻る |

---

## 4. 操作・イベント

| 操作 | 内容 |
| -- | -- |
| 画面初期表示 | 各ボタンを表示 |
| 単語管理ボタン押下 | S-03 へ遷移 |
| クイズモードボタン押下 | S-06 へ遷移 |
| ログアウトボタン押下 | AUTH-02を実行後、S-01へ遷移 |

---

## 5. バリデーション

なし

---

## 6. エラーハンドリング

| ケース | 表示内容 |
| -- | -- |
| セッション無効 | S-01ログイン画面へリダイレクト |
| API通信失敗 | 汎用エラーメッセージを表示 |

---

## 7. 画面遷移

| 操作 | 遷移先 |
| -- | -- |
| 単語管理 | S-03 単語一覧画面 |
| クイズモード | S-06 クイズ画面 |
| ログアウト | S-01 ログイン画面 |

---

## 8. 備考

- ログイン状態でのみアクセス可能
- 本画面は全機能画面から遷移可能とする
