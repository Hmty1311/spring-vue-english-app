# S-03 単語一覧画面 基本設計書

---

## 1. 画面概要

ログインユーザーが登録した英単語を一覧表示する画面。
検索・暗記状態による絞り込み・ページングを行い、
学習および単語管理の起点となる画面。

---

## 2. 利用API

| API ID | API名 | 用途 |
| --- | -- | -- |
| WORD-01 | 単語一覧取得 | 一覧表示・検索・絞り込み |
| WORD-04 | 単語更新 | 暗記済みフラグ切り替え |

---

## 3. 表示項目

### 3-1. ヘッダー部（検索エリア）

| 表示名 | 項目名 | 表示 | 備考 |
| -- | -- | -- | -- |
| 検索キーワード | keyword | ○ | 部分一致検索 |
| 検索対象 | searchTarget | ○ | 英単語/日本語訳 |
| 暗記状態 | memorizedFilter | ○ | 全て/未暗記/暗記済み |
| メニューへ戻るボタン | backToMenuButton | ○ | S-02へ遷移 |

### 3-2. 明細部（一覧エリア）

| 表示名 | 項目名 | 表示 | 備考 |
| -- | -- | -- | -- |
| 英単語 | word | ○ | 主要情報 |
| 意味 | meaning | ○ | 日本語訳 |
| 暗記済み | memorized | ○ | チェックボックス |

※英単語クリックで単語詳細画面（S-05）へ遷移

### 3-3. フッター部

| 表示名 | 項目名 | 表示 | 備考 |
| -- | -- | -- | -- |
| ページ番号リンク | pageNumberLink | ○ | 押下で該当ページを表示 |
| 現在ページ表示 | currentPage | ○ | 現在ページ番号 |
| 総ページ数 | totalPages | ○ | 総ページ数表示 |

---

## 4. 操作・イベント

| 操作 | 内容 |
| -- | -- |
| 新規登録ボタン押下 | 単語登録画面（S-04）へ遷移 |
| 検索条件変更 | 一覧を再取得（ページは1にリセット） |
| 暗記チェック切替 | WORD-04を呼び出し暗記状態を更新 |
| ページ番号押下 | 対象ページの一覧を取得 |
| 英単語クリック | 単語詳細画面（S-05）へ遷移 |

---

## 5. 検索・絞り込み条件

| 条件 | UI | 内容 |
| -- | -- | -- |
| 検索キーワード | テキスト入力 | 部分一致検索 |
| 検索対象 | ラジオボタン | 英単語/日本語訳のいずれか |
| 暗記状態 | 切り替え | 全て/未暗記/暗記済み |

※検索対象（英単語・意味）は同時指定不可

---

## 6. ページング

| 項目 | 内容 |
| -- | -- |
| 1ページ件数 | 最大50件 |
| ページング方式 | ページ番号指定 |
| 表示 | 現在ページ/総ページ数表示 |

---

## 7. エラーハンドリング

| ケース | 表示内容 |
| --- | --- |
| 一覧取得失敗 | エラーメッセージ表示 |
| 該当データなし | 「該当する単語がありません」表示 |

---

## 8. 画面遷移

| 操作 | 遷移先 |
| -- | -- |
| 新規登録ボタン押下 | S-04 単語登録画面 |
| 英単語クリック | S-05 単語詳細画面 |
| メニューへ戻るボタン押下 | S-02 モード選択画面 |


---

## 9. 備考

 - 検索対象（英単語/意味）は同時指定不可
 - 暗記済みフラグは一覧上で直接更新可能
 - ページサイズは固定（50件）とする
 - 新規登録は専用画面（S-04）で行う
 - 登録完了後は一覧画面を再表示したタイミングで最新データを取得する
