# S-05 単語詳細画面 基本設計書

---

## 1. 画面概要

選択された英単語の詳細情報を表示・編集する画面。
英単語・意味・例文・暗記済みフラグを確認・変更できる。
更新および削除操作を提供し、削除後は一覧画面へ遷移する。

---

## 2. 利用API

| API ID | API名 | 用途 |
| --- | -- | -- |
| WORD-02 | 単語詳細取得 | 初期表示 |
| WORD-04 | 単語更新 | 内容更新 |
| WORD-05 | 単語削除 | 単語削除 |

---

## 3. 表示項目

### 3-1. 入力項目

| 表示名 | 項目名 | 表示 | 備考 |
| -- | -- | -- | -- |
| 英単語 | word | ○ | 編集可・必須 |
| 意味 | meaning | ○ | 編集可・必須 |
| 例文 | example | ○ | 編集可・任意 |
| 暗記済み | memorized | ○ | チェックボックス |

※初期表示時はWORD-02のレスポンスを表示する

---

## 4. 操作・イベント

| 操作 | 内容 |
| -- | -- |
| 更新ボタン押下 | 入力内容を送信し、WORD-04を呼び出す |
| 更新成功 | 成功メッセージ表示 |
| 削除ボタン押下 | 確認ダイアログ表示 |
| 削除確定 | WORD-05を呼び出し、一覧画面（S-03）へ遷移 |
| 戻るボタン押下 | 変更がある場合、確認ダイアログを表示 |
| 戻る確定 | 単語一覧画面（S-03）へ遷移 |

---

## 5. バリデーション

| 項目 | 内容 |
| -- | -- |
| word | 必須、1～100文字 |
| meaning | 必須、1～255文字 |
| example | 任意 |
| memorized | 任意 |

---

## 6. エラーハンドリング

| ケース | 表示内容 |
| -- | -- |
| 詳細取得失敗 | エラーメッセージ表示 |
| バリデーションエラー | 入力項目ごとにエラーメッセージ表示 |
| 更新失敗 | フォームを保持したまま、エラーメッセージ表示 |
| 削除失敗 | 一覧へ遷移せず、エラーメッセージ表示 |
| 戻る操作時 | 確認ダイアログ表示 |

---

## 7. 画面遷移

| 操作 | 遷移先 |
| -- | -- |
| 戻るボタン押下 | S-03 単語一覧画面 |
| 削除成功 | S-03 単語一覧画面 |

---

## 8. 備考

- 本画面はS-03からの遷移のみを想定
- 初期表示時は必ず単語IDをPath Parameterで受け取る
- 暗記済みフラグは単独更新せず、更新ボタン押下時にまとめて更新
- 削除操作は確認ダイアログを表示した上で実行する
- 削除確認ダイアログの文言は「削除します。よろしいですか？」とする
- 削除成功後は一覧画面で最新データを再取得する
- 保存の変更がある状態で戻る操作を行った場合は、
  「変更内容は破棄されますが、よろしいですか？」を表示する
