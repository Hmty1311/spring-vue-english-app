# S-05 単語詳細画面 基本設計書

---

## 1. 画面概要

登録済みの英単語1件の詳細情報を表示・編集する画面。
ログインユーザーが登録した単語のみを対象とする。
単語の内容確認、編集、削除を行う。

---

## 2. 利用API

| API ID | API名 | 用途 |
| --- | -- | -- |
| WORD-02 | 単語詳細取得 | 初期表示 |
| WORD-04 | 単語更新 | 内容更新 |
| WORD-05 | 単語削除 | 単語削除 |

---

## 3. 表示項目

### 3-1. 表示/入力項目

| 表示名 | 項目名 | 表示 | 備考 |
| -- | -- | -- | -- |
| 英単語 | word | ○ | 編集可・必須 |
| 意味 | meaning | ○ | 編集可・必須 |
| 例文 | example | ○ | 編集可・任意 |
| 暗記済み | memorized | ○ | チェックボックス |
| 登録日時 | createdAt | ○ | 表示のみ |
| 更新日時 | updatedAt | ○ | 表示のみ |

---

## 4. 操作・イベント

| 操作 | 内容 |
| -- | -- |
| 画面表示時 | WORD-02を呼び出し |
| 更新ボタン押下 | 入力内容を送信し、WORD-o4を呼び出す |
| 削除ボタン押下 | 確認ダイアログ表示後、WORD-05を呼び出す |
| 戻るボタン押下 | 単語一覧画面（S-03）へ戻る |

---

## 5. バリデーション

| 項目 | 内容 |
| -- | -- |
| word | 必須、1～100文字 |
| meaning | 必須、1～255文字 |
| example | 任意 |
| memorized | 任意 |

---

## 6. エラーハンドリング

| ケース | 表示内容 |
| -- | -- |
| データ取得失敗 | エラーメッセージ表示 |
| 更新失敗 | エラーメッセージ表示 |
| 削除失敗 | エラーメッセージ表示 |

---

## 7. 備考

- 編集内容は明示的に「更新」操作を行った場合のみ反映される
- 削除操作は確認ダイアログを必須とする
- タグ機能は使用しない
- URLは/words/{wordId}を想定
