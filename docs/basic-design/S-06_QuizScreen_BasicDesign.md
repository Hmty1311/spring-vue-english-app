# S-06 クイズ画面 基本設計書

---

## 1. 画面概要

登録されている英単語を用いたクイズを出題する画面。英単語に対する意味を回答し、正誤結果を確認できる。1回のクイズにつき、ランダムに選択された10問を1問ずつ出題する。解凍後に次の問題へ進行する。

---

## 2. 利用API

| API ID | API名 | 用途 |
| --- | -- | -- |
| WORD-06 | クイズ用単語取得 | 出題用単語取得 |

---

## 3. 表示項目

### 3-1. 表示項目一覧

| 表示名 | 項目名 | 表示 | 備考 |
| --- | -- | -- | -- |
| 出題英単語 | word | ○ | 編集不可 |
| 回答入力欄 | answer | ○ | テキスト入力・必須 |
| 正誤結果 | result | △ | 回答後に表示 |
| 正解の意味 | meaning | △ | 不正解時のみ表示 |
| 進捗数 | progress | ○ | 例：3/10 |

---

## 4. 操作・イベント

| 操作 | 内容 |
| -- | -- |
| 画面初期表示 | WORD-06を呼び出し、1問目を表示 |
| 回答ボタン押下 | 入力値と正解を比較し、正誤判定 |
| 正解 | 正解メッセージ表示 |
| 不正解 | 不正解メッセージと正しい意味を表示 |
| 次へボタン押下 | 次の問題を表示 |
| 最終問題回答後 | 結果画面（S-07） |
| 戻るボタン押下 | 単語一覧画面（S-03） |

---

## 5. バリデーション

| 項目 | 内容 |
| -- | -- |
| answer | 必須、1～255文字 |

---

## 6. エラーハンドリング

| ケース | 表示内容 |
| -- | -- |
| 出題取得失敗 | エラーメッセージ表示 |
| 回答未入力 | 入力エラーメッセ－ジ表示 |

---

## 7. 画面遷移

| 操作 | 遷移先 |
| -- | -- |
| 戻るボタン押下 | S-03 単語一覧画面 |
| クイズ完了 | S-07 学習結果画面 |

---

## 8. 備考
- クイズ対象は暗記済みフラグがOFFの単語を優先する
- 出題数は1回につきランダムに選択された10問とする
- 出題順はランダムとする
- 正誤判定はフロントエンドで行う
- 回答結果はサーバーへ送信しない
- クイズ途中で戻る操作を行った場合、進捗は破棄される
