# S-03 単語一覧画面 基本設計書

---

## 1. 画面概要

登録されている英単語を一覧表示する画面。ログインユーザーが登録した単語のみを対象とし、検索・ページング・詳細画面への遷移を行う。学習の起点となる画面。

---

## 2. 利用API

| API ID | API名 | 用途 |
| --- | -- | -- |
| WORD-01 | 単語一覧取得 | 一覧表示・検索・絞り込み |
| WORD-04 | 単語更新 | 暗記済みフラグ切り替え |

---

## 3. 表示項目

### 3-1. 一覧表示項目

| 表示名 | 項目名 | 表示 | 備考 |
| -- | -- | -- | -- |
| 英単語 | word | ○ | 主要情報 |
| 意味 | meaning | ○ | 日本語訳 |
| 例文 | example | - | 一覧では不要 |
| 暗記済み | memorized | ○ | チェックボックス |

※英単語クリックで単語詳細画面（S-05）へ遷移

---

## 4. 検索・絞り込み条件

### 4-1. 検索条件

| 条件 | UI | 内容 |
| -- | -- | -- |
| 検索キーワード | テキスト入力 | 部分一致検索 |
| 検索対象 | ラジオボタン | 英単語/日本語訳のいずれか |
| 暗記状態 | 切り替え | 全て/未暗記/暗記済み |

---

## 5. ページング

| 項目 | 内容 |
| -- | -- |
| 1ページ件数 | 最大50件 |
| ページング方式 | ページ番号指定 |
| 表示 | 現在ページ/総ページ数表示 |

---

## 6. 操作・イベント

| 操作 | 内容 |
| -- | -- |
| 新規登録ボタン押下 | 単語登録画面（S-04）へ遷移 |
| 検索条件変更 | 一覧再取得（ページは1にリセット） |
| 暗記チェック切替 | 随時更新（WORD-04呼び出し） |
| ページ切替 | 対象ページの一覧取得 |

---

## 7. エラーハンドリング

| ケース | 表示内容 |
| --- | --- |
| データ取得失敗 | エラーメッセージ表示 |
| データ0件 | 「該当する単語がありません」表示 |

---

## 8. 備考
 - 検索対象（英単語/意味）は同時指定不可
 - 暗記済みフラグは一覧上で直接更新可能
 - ページサイズは固定（50件）とする
 - 新規登録は専用画面（S-04）で行う
 - 登録完了後は一覧画面を再表示したタイミングで最新データを取得する
