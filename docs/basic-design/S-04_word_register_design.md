# S-04 単語登録画面 基本設計書

---

## 1. 画面概要

英単語を新規登録する画面。
入力された単語はログインユーザーに紐づいて登録される。

---

## 2. 利用API

| API ID | API名 | 用途 |
| --- | -- | -- |
| WORD-03 | 単語登録 | 単語の新規登録 |

---

## 3. 表示項目

### 3-1. 一覧表示項目

| 表示名 | 項目名 | 表示 | 備考 |
| -- | -- | -- | -- |
| 英単語 | word | ○ | 必須入力 |
| 意味 | meaning | ○ | 必須入力 |
| 例文 | example | ○ | 任意入力 |
| 暗記済み | memorized | ○ | チェックボックス（初期値：未暗記） |

---

## 4. 操作・イベント

| 操作 | 内容 |
| -- | -- |
| 登録ボタン押下 | 入力内容を送信し、WORD-03を呼び出す |
| 登録成功 | 成功メッセージ表示後、入力フォームを初期化 |
| 戻るボタン押下 | 単語一覧画面(S-03)へ戻る |

※自動遷移は行わない

---

## 5. バリデーション

| 項目 | 内容 |
| -- | -- |
| word | 必須、1~100文字 |
| meaning | 必須、1~255文字 |
| example | 任意 |
| memorized | 任意(未指定時はfalse) |

---

## 6.エラーハンドリング

| ケース | 表示内容 |
| -- | -- |
| バリデーションエラー | 入力項目ごとにエラーメッセージ表示 |
| 登録失敗 | フォームは保持したまま、共通エラーメッセージ表示 |

---

## 7. 備考
- 登録成功後は画面遷移せず、連続登録可能とする
- 登録成功時は以下を行う
  - 入力フォームを初期化
  - 暗記済みフラグは初期化（未暗記）に戻す
- 登録時に暗記済みフラグを指定可能
- タグ機能は使用しない
- 一覧反映は、一覧画面再表示時に再取得する
