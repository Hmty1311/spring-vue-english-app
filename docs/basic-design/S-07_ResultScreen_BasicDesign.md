# S-07 学習結果画面 基本設計書

---

## 1. 画面概要

クイズ画面（S-06）で実施した学習結果を表示する画面。全10問に対する正解数・正答率を確認でき、再度クイズを行う、または単語一覧画面へ戻ることができる。

---

## 2. 利用API

本画面ではAPI通信は行わない。（表示内容はS-06で保持している結果情報を引き継ぐ）

---

## 3. 表示項目

### 3-1. ヘッダー部

| 表示名 | 項目名 | 表示 | 備考 |
| --- | --- | -- | -- |
| メニューへ戻るボタン | backToMenuButton | ○ | S-02へ遷移 |

### 3-2. 明細部

| 表示名 | 項目名 | 表示 | 備考 |
| --- | --- | -- | -- |
| 正解数 | correctCount | ○ | 例：7問正解 |
| 出題数 | totalCount | ○　| 固定値：10 |
| 正答率 | accuracy | ○ | 例：70% |
| 結果メッセージ | resultMessage | ○ | 正答率に応じて表示内容を変更  |

### 3-3. フッター部

| 表示名 | 項目名 | 表示 | 備考 |
| --- | --- | -- | -- |
| もう一度ボタン | retryButton | ○ | S-06へ遷移 |
| 一覧へ戻るボタン | backToListButton | ○ | S-03へ遷移 |

---

## 4. 操作・イベント

| 操作 | 内容 |
| -- | -- |
| 画面初期表示 | クイズ結果を表示 |
| もう一度ボタン押下 | クイズ画面（S-06）へ遷移 |
| 一覧へ戻るボタン押下 | 単語一覧画面（S-03）へ遷移 |

---

## 5. バリデーション

なし

---

## 6. エラーハンドリング

| ケース | 表示内容 |
| -- | -- |
| 結果情報が存在しない | 単語一覧画面（S-03）へリダイレクト |

---

## 7. 画面遷移

| 操作 | 遷移先 |
| -- | -- |
| もう一度 | S-06 クイズ画面 |
| 一覧へ戻る | S-03 単語一覧画面 |
| メニューへ戻るボタン押下 | S-02 モード選択画面 |

---

## 8. 備考

- 出題数は常に10問とする
- 正答率は「正解数 / 出題数 * 100」で算出する
- 結果メッセージは以下の基準で表示する
    - 正答率80%以上：非常によくできました
    - 正答率50%以上80%未満：よくできました
    - 正答率50%未満：復習しましょう
- 本画面は直接遷移不可とし、S-06からのみ遷移可能とする
